# Config Files

## Special Field Descriptions

The following fields are typically overwritten at runtime and should be kept empty in sample configuration files:

| Field              | Description                 |
| ------------------ | --------------------------- |
| `cwd`              | Working directory           |
| `sessionId`        | Session ID                  |
| `version`          | Version number              |
| `gitBranch`        | Git branch name             |
| `userType`         | User type                   |
| `isSidechain`      | Whether this is a sidechain |
| `thinkingMetadata` | Thinking metadata           |
| `todos`            | Todo items                  |
